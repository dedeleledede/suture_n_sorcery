# Suture & Sorcery

A Fabric mod for Minecraft **1.21.10** that explores blood, thread, and ritual mechanics with custom blocks, items, effects, and shaders.

> **Status:** In active development.

---

## Requirements
- Minecraft **1.21.10**
- Fabric Loader (latest for 1.21.10)
- Fabric API (matching 1.21.10)
